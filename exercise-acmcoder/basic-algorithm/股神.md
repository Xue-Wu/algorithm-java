# Basic algorithm
## 示例代码
``` java
import java.io.*;
import java.util.*;
public class Main
{
    public static void main(String args[])
    {
        Scanner cin = new Scanner(System.in);
      	// 读取需要第几天的数据
        int day = cin.nextInt();
      	// 初始股价为1
      	int price = 1;
      	int dummyIncrease = 2;
        // 需要涨价的天数
      	int increase = 1;
      	// 从第二天开始算起
      	for (int i = 2; i <= day; i++) {
			if (increase != 0) {
              	// 处理涨价的情况
            	price++;
              	increase--;
            } else {
              	// 处理减价的情况
            	price--;
              	increase = dummyIncrease;
              	dummyIncrease++;
            }
        }
      	System.out.println(price);
    }
}
```
