# Jump Game II
## 示例代码
``` java
class Solution {
    public int jump(int[] nums) {
        int length = nums.length;
        // cur表示当前能到达的最远位置
        // pre表示之前能到达的最远位置
        int cur = 0, pre = 0, res = 0, i = 0;

        while (cur < length - 1) {
            res++;
            pre = cur;
            for (; i <= pre; i++) {
                cur = Math.max(cur, i + nums[i]);
            }
        }
        return res;
    }
}
```
